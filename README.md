# 🔐 Login & Registration System (Java Servlets + JSP)

## 📌 Overview
This project is a web-based authentication system built using Java Servlets, JSP, and JDBC following MVC architecture.

## 🚀 Features
- User Registration
- User Login Authentication
- Session Management
- Logout Functionality
- Protected Pages (Session-based access)

## 🛠️ Tech Stack
- Java (Servlets & JSP)
- JDBC
- MySQL
- HTML, CSS

## 🧠 Architecture
MVC Pattern:
- Controller → Servlets
- Model → User class
- DAO → Database operations
- View → JSP & HTML

## 🔄 Workflow
1. User registers → data stored in database  
2. User logs in → credentials validated  
3. Session created → access granted  
4. Logout → session destroyed  

## 🗄️ Database
Table: AccountDetails

Fields:
- fname
- lname
- email
- dob
- pass
